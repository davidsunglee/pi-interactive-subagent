## Goal
Ensure async subagents return their transcript bodies to the main pi agent, and that rendered transcript content uses the correct background color.

## Context
Async subagent results currently need to surface the transcript body back in the main pi agent. The UI/background styling should match the expected transcript/background color instead of inheriting or rendering with the wrong background.

## Acceptance criteria
- Async subagent completion messages include the transcript body/content needed by the main pi agent.
- The main pi agent receives and can display the returned transcript body for async subagents.
- Transcript body rendering uses the correct background color in the main agent UI.
- Add or update tests/fixtures covering async subagent result payloads and background color rendering if applicable.
- Verify existing sync subagent behavior is not regressed.
